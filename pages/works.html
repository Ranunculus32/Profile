Twitter Clone

This is a group project presented on May 22, 2024. We are tasked to clone twitter. The landing page seen here is an exact hand coded copy of X landing page. Due to dual servers, this image is not clickable which leads to the project itself. Intead, I will explain what the project is all about.

The functionalities me and my team managed to implement in this project are: user registration, login and logout, posting of tweets, following a page or profile, search by username, liking a tweet and nested comments or replies.

The stack used are: MongoDb, React, Node.js, Javascript, Bootstrap and css.
